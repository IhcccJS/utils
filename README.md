## Installation

Install `@ihccc/utils` with npm

```bash
  pnpm i @ihccc/utils
```

## Usage/Examples

```javascript
import { uuid } from '@ihccc/utils'

function foo() {
  const id = uuid();
  // ...
}
```

## Run Locally

Clone the project

```bash
  git clone https://@ihccc/utils
```

Go to the project directory

```bash
  cd utils
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```

## Running Tests

To run tests, run the following command

```bash
  pnpm run test
```


## Deployment

To deploy this project run

```bash
  pnpm run deploy
```
